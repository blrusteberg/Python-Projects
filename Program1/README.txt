Python3 and above must be downloaded to run these 2 files!!

1) In terminal type 'make' in the same folder the server.py, client.py and makefile are in.

2) Open another terminal and locate the server.py file

3) Type './server.py <server-port> (1-65535) and in the first terminal type 
   './client.py <hostname of server> <port> EX: ./client.py blruste 60000

4) Follow the instructions in the client.py script. 

NOTE - To close the client type BYE '<hostname>' at any time.
       To close the server type CTRL + Z.
